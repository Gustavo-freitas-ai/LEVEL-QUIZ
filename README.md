## 💻 Sobre o projeto

Este projeto consiste em um quiz por níveis, onde cada fase se torna progressivamente mais difícil. Atualmente, o sistema conta com 3 níveis implementados, cada um com perguntas de dificuldade crescente.

O projeto foi desenvolvido como um projeto pessoal de aprendizado, com foco em aprofundar conhecimentos em Python e na biblioteca Flask, além de praticar conceitos de lógica de programação, estruturação de rotas e design de interface.

O principal objetivo foi explorar novas funcionalidades do Python, entender melhor o funcionamento do Flask no desenvolvimento web e aprimorar habilidades de organização de código e estilização.

🚀 Projeto voltado para prática, evolução técnica e experimentação.
